# eslint-plugin-fsd-rules-plugin
---

## 💿 Install

```
npm install --save-dev eslint-plugin-fsd-rules-plugin
```

## 📖 Rules

| Name                | Description                                                     |
| ------------------- | --------------------------------------------------------------- |
| path-checker        | Checking file imports from a single module                      |
| public-api-imports  | Absolute import is allowed only from the public api (index.ts)  |
